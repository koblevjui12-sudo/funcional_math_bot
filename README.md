# Математический Telegram-бот

Программное решение для автоматизации математических расчетов, вычисления налоговых ставок и визуализации функций. Проект разработан с учетом особенностей ввода данных на мобильных устройствах.

## Функциональные возможности

1. **Налоговые исчисления 2025**
   * Расчет НДФЛ по прогрессивной шкале (13%, 15%, 18%, 20%).
   * Вывод итоговой суммы налога и чистого дохода после вычета.

2. **Математический калькулятор**
   * Поддержка операций с обыкновенными дробями.
   * Вывод результата в трех форматах: обыкновенная, смешанная и десятичная дробь.
   * Обработка спецсимволов степени (квадрат, куб) и автоматическая подстановка знаков умножения.

3. **Визуализация графиков**
   * Построение графиков функций в стандартном диапазоне координат.
   * Автоматическое распознавание пользовательского ввода формул (поддержка "y=" и слитого написания).

4. **Статистический анализ**
   * Вычисление среднего арифметического значения для произвольного числового набора.

## Технический стек

* **Язык программирования:** Python 3
* **Библиотека Telegram API:** Aiogram 3.x
* **Математический движок:** Simpleeval, Fractions
* **Графический движок:** Matplotlib, Numpy
